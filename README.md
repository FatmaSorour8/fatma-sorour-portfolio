## 📋 Overview

A modern, responsive portfolio website showcasing my work as a Full-Stack Developer. Built with a focus on performance, accessibility, and user experience. The portfolio highlights my skills, experience, projects, and provides a way for potential employers or clients to get in touch.

## ✨ Features

- 🎨 **Dark/Light Mode** - Automatic theme switching with persistent user preference
- 📱 **Fully Responsive** - Optimized for all device sizes (mobile, tablet, desktop)
- ⚡ **Smooth Animations** - Scroll-triggered reveals and micro-interactions
- 🖼️ **Dynamic Canvas Background** - Interactive network particle animation
- 📧 **Working Contact Form** - Send messages directly via FormSubmit.co
- ⌨️ **Typewriter Effect** - Animated text in hero section
- 🎯 **SEO Optimized** - Meta tags, structured data, and semantic HTML
- ♿ **Accessibility Ready** - Proper ARIA labels and keyboard navigation
- 🚀 **Performance Optimized** - Fast loading with minimal dependencies

## 🛠️ Technologies Used

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Custom properties, animations, flexbox, grid
- **JavaScript (ES6+)** - DOM manipulation, async/await, Intersection Observer
- **Tailwind CSS** - Utility-first CSS framework
- **Font Awesome 6** - Icon library

### External Services
- **FormSubmit.co** - Email handling without backend
- **Google Fonts** - Roboto Mono typography

### Tools & APIs
- **Canvas API** - Particle network animation
- **Intersection Observer API** - Scroll-triggered animations
- **LocalStorage API** - Theme persistence
